## CSS选择器
CSS选择器分为如下几种类别：
* 简单选择器
  * 标签选择器
  * 类选择器
  * id选择器
  * 通配符选择器
  * 属性选择器
  * 伪类选择器
* 其他选择器
  * 伪元素选择器
  * 组合选择器
  * 选择器分组
### 简单选择器
简单选择器可组合使用
#### 标签选择器
```
<div>
<p>Sample Paragraph</p>
<p>Sample Paragraph</p>
<p>Sample Paragraph</p>
</div>
<style type="text/css">
p {
color: blue;
}
</style>
```

参考：https://developer.mozilla.org/zh-CN/docs/Web/Guide/CSS/Getting_Started/Selectors
